# Anti-Drift Prompt Rules

## Always Include at Least 4 of These

- black velvet background
- fluorescent / blacklight palette
- airbrushed glow
- fantasy / mystical / cosmic iconography
- iconic poster readability
- soft bloom halo
- decorative stars / aura haze

## If the Result Looks Too Synthwave

Add:
- mystical fantasy wall poster
- painterly airbrush
- dream-animal energy
- occult celestial atmosphere

Remove or reduce:
- grid
- chrome car
- retro sunset
- outrun city

## If the Result Looks Too Cute / Sticker-Like

Add:
- black velvet void
- deeper shadow
- mysterious atmosphere
- dramatic contrast
- occult aura

## If the Result Looks Too Digital-Clean

Add:
- painterly
- airbrush
- slight pigment bloom
- soft overspray
- analog poster texture

## If the Result Looks Too Muddy

Add:
- stronger black field
- fluorescent magenta
- electric blue
- cyan highlights
- sharper glow hierarchy
