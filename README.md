# Vintage 1970s–1980s Blacklight Poster Style Repo

A modular style repo for recreating and extending the visual language of vintage 1970s–1980s
blacklight posters for image prompting, shader design, generative art systems, motion graphics,
visual engines, and hybrid digital tools.

This repo treats the style as a reusable visual system rather than a loose vibe label.

## Core Idea

Vintage blacklight poster aesthetics are built from:

- near-total black backgrounds
- fluorescent or Day-Glo color logic
- high-contrast glow emission illusions
- airbrushed, velvet-like transitions
- fantasy, cosmic, mystical, animal, and erotic-adjacent iconography
- iconic across-the-room readability
- theatrical, non-naturalistic color relationships

This is not generic neon.
It is not generic psychedelic art.
It is not synthwave.
It is not Lisa Frank.
It is its own visual machine.

## Repo Goals

This repo is designed to support:

- AI image prompting
- shader development
- generative visual tools
- motion graphics systems
- style transfer systems
- design language extraction
- visual vocabulary hoarding
- anti-drift style enforcement

## Folder Guide

### `/docs`
High-level theory, historical cues, boundaries, use cases, and stylistic identity.

### `/systems`
Operational rules for palette, motifs, optics, composition, textures, typography, and recurring
poster artifacts.

### `/prompts`
Reusable prompt scaffolds, modular style blocks, and anti-drift prompt control.

### `/shader`
Translation of the style into glow systems, emission stacks, noise logic, bloom hierarchy, and
parameter mappings.

### `/data`
Machine-readable files for apps, scripts, engines, and automated style workflows.

### `/examples`
Example poster build recipes using the repo’s internal logic.

7. Use `data/` for app logic and automation

## Design Philosophy

Treat blacklight poster art as a system of controlled fluorescence against darkness.

The image should feel less like a picture of a thing and more like a hallucinated icon glowing in
a void.
