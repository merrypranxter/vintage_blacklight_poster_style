# Neighbor Style Comparisons

## Compared to Psychedelic Concert Posters

Blacklight posters:
- more fluorescent
- more painterly / airbrushed
- less typography-dominant
- more wall-decor icon logic
- more dream-animal / fantasy imagery

Concert posters:
- more graphic
- more layout-driven
- more text interplay
- often flatter print logic

## Compared to Synthwave

Blacklight posters:
- more organic
- more mystical
- more airbrushed
- more fantasy / occult / animal / cosmic

Synthwave:
- more geometric
- more digital-clean
- more retro-tech
- more horizon / grid / sports car imagery

## Compared to Fantasy Paperback Covers

Blacklight posters:
- more iconically simplified
- more glow-driven
- more black-dominant
- more decorative and aura-centered

Fantasy covers:
- more narrative scene-based
- more conventional illustration lighting
- more environment-specific detail

## Compared to Lisa Frank

Blacklight posters:
- darker
- more mysterious
- more erotic or spiritual
- more velvet and aura-driven

Lisa Frank:
- brighter overall
- cuter
- glossier
- more school-supply commercial polish

## Compared to Velvet Paintings

Blacklight posters share:
- black velvet void logic
- luminous contrast
- fantasy animals
- iconic center-focused compositions

But blacklight posters usually push:
- more fluorescent chemistry
- more UV-reactive palette logic
- more cosmic and psychedelic aura behavior
